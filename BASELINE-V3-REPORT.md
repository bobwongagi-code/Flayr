## 核心发现（诚实，不 spin）
1. **抖动主旋律 = 执行分 B/C 翻档（轴B 地盘），非大边界漂移。** 12 个⚠ 全是"severity 稳/B-C 跳"；10 个🔴 多数也是 B 或 C 在相邻档跳。模型对"这侧到位没"的 0/0.5/1/2 判断，重跑间到处飘。
2. **🔴 弥漫、不集中。** 每阶段都至少 1 个🔴（S3 最多 3 个）。系统性分数不稳，不是某格坏。
3. **第三个抖源**：are_xie S5 **B/C 都稳（5/5）但 severity 翻**——derive 政策输入（痛点命中/背书正则）也抖（§0）。
4. **极端不稳样本**：youkoubo-c2（糊弄）连合法输出都凑不齐；wukoubo-c0（无口播）QA 失败率高。无口播/糊弄=最不稳簇。

## 边界 + 对子②的含义
- **本数据无法干净分轴A vs 轴B**（轴A=同段 functions 归属一致度，需 functions 取证，本数据 pre-functions）。
- 但强烈暗示：**轴B（执行分/感知翻档）是大头且弥漫；functions（治轴A/归属）不直接碰执行分**。回应"弄完轴A 轴B 还用不用"——本数据偏向"轴B 是真问题、可能比轴A 大"。
