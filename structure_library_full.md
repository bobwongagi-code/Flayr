# Chimera 带货短视频结构库 v1.0（全量版）

> 本文档包含所有 32 个模块的完整素材需求清单（镜头/文案/声音/节奏四维）、降级规则、适配条件。
> 压缩版（`structure_library.md`）用于 Agent 上下文输入，本文档用于人工参考和按需检索。

---

## 一、骨架：3 段 6 槽位

| 槽位 | 职责 | 时长占比 | 可跳过 |
|------|------|----------|--------|
| S1 Hook | 抢夺注意力，让用户留下 | 10~20% | 否 |
| S2 产品引出 | 从 Hook 自然过渡到产品 | 5~10% | 否（可与 S3 合并） |
| S3 使用过程 | 展示"怎么用"，建立真实感 | 20~25% | 否 |
| S4 效果呈现 | 展示"用了之后怎样"，建立说服力 | 20~25% | 否 |
| S5 信任放大 | 用对比/证言/数据放大可信度 | 10~15% | 是 |
| S6 CTA | 临门一脚，推动下单 | 10~20% | 否 |

**S2+S3 合并条件**：成片目标时长 ≤ 15s 时自动合并。

**S5 跳过条件**：品类属于低决策快消（食品饮料、家清日用）且成片 ≤ 15s 且购买动机 = MO-刚需补货。

## 二、标签体系（12 个）

### 商品侧

| 编号 | 标签 | 取值 | 来源 |
|------|------|------|------|
| T1 | 品类 | 29 项枚举（见附录 A） | 商品字段 |

### 素材侧

| 编号 | 标签 | 取值 | 判断方式 |
|------|------|------|----------|
| T2 | 素材丰富度 | MR-1(<60s) / MR-2(60~300s) / MR-3(300~900s) / MR-4(>900s) | sum(clip.duration) |
| T3 | 人像素材 | bool | 视频理解检测人脸/人体 |
| T4 | 对比素材 | bool | 检测前后状态素材 |
| T5 | 口播素材 | bool | 检测人声+口型匹配 |
| T6 | 产品特写 | bool | 检测产品近距离镜头 |

### 人群侧

| 编号 | 标签 | 取值 | 来源 |
|------|------|------|------|
| T7 | 性别倾向 | G-F / G-M / G-N | 品类映射 |
| T8 | 年龄段 | A1(18~25) / A2(26~35) / A3(36~50) / A4(50+) | 品类映射 |
| T9 | 购买动机 | MO-解决问题 / MO-提升体验 / MO-情感满足 / MO-刚需补货 | 品类+关键词 |

## 三、素材需求清单统一字段结构

每个模块的素材清单均按以下四维结构描述：

```
【镜头】景别 / 主体 / 镜头动作 / 时长 / 数量 / 必需or可选
【文案】类型 / 句式模板 / 字数范围 / 语气
【声音】口播 / BGM 情绪 / 音效
【节奏】剪辑节奏 / 卡点
```

## 四、S1 Hook 模块库（7 个模块）

### S1-A：痛点提问型

用"你是不是也……""为什么你的 XX 总是……"直接戳问题。

```
【适配条件】
  必需素材：T3=true, T5=true
  适配品类：通用
  排除品类：无
  适配购买动机：MO-解决问题

【镜头】
  景别：近景 + 特写（痛点细节）
  主体：人（面部表情为主）+ 问题场景（头发掉落/污渍/肥肉等）
  镜头动作：静态或轻微推近
  时长：单镜头 1~2 秒
  数量：2~3 个镜头（人脸 1 + 痛点细节 1~2）
  必需：人像素材 T3 = true

【文案】
  类型：字幕 + 口播
  句式模板："你是不是也……""为什么 XX 总是……""有没有发现……"
  字数范围：8~15 字
  语气：疑问

【声音】
  口播：必需（第一人称提问）
  BGM 情绪：紧张或悬疑（低音量铺底）
  音效：可选心跳/滴答

【节奏】
  剪辑节奏：中速
  卡点：不需要

【降级规则】
  T3=false 时，降级到"画面文字+口播"纯文字 Hook，用黑底白字+口播提问替代人脸镜头。
```

---

### S1-B：反差震惊型

用视觉或结果上的强反差制造意外感。

```
【适配条件】
  必需素材：T4=true
  适配品类：通用
  排除品类：无
  适配购买动机：通用

【镜头】
  景别：特写或中景
  主体：产品 + 对比物（或前后两个状态）
  镜头动作：快速切换，建议一刀切
  时长：单镜头 0.5~1 秒
  数量：2 个镜头（A 状态 + B 状态）
  必需：T4 对比素材 = true

【文案】
  类型：画面文字（大字压屏）
  句式模板："XX 秒前 / XX 秒后""没想到 XX 这么 XX"
  字数范围：4~8 字
  语气：惊讶

【声音】
  口播：可选
  BGM 情绪：高燃或悬疑
  音效：必需（破碎音/震撼音/whoosh）

【节奏】
  剪辑节奏：快切
  卡点：建议对齐音效鼓点

【降级规则】
  T4=false 时，此模块不可用，从候选集中剔除。
```

### S1-C：结果前置型

【适配条件】

  排除品类：虚拟商品、二手、收藏品
  适配购买动机：通用

【镜头】

  主体：使用后的效果状态
  镜头动作：静态或慢推
  时长：1~2 秒
  数量：1~2 个镜头

【文案】
  类型：画面文字
  句式模板："用完就这样""XX 天后变成这样"
  字数范围：5~10 字
  语气：肯定

【声音】
  口播：可选
  BGM 情绪：轻快或悬疑
  音效：可选 ding 声

【节奏】
  剪辑节奏：中速起手
  卡点：不需要

【降级规则】
  T6=false 时，降级到"产品正脸静态图+画面文字"极简形式。
```

### S1-E：悬念故事型

用一个未完成的小剧情勾住用户，产品藏在故事里。

```
【适配条件】
  必需素材：T3=true, T2≥MR-2
  适配品类：通用
  排除品类：无
  适配购买动机：通用

【镜头】
  景别：中景 + 近景交替
  主体：人（剧情演员）+ 情节道具
  镜头动作：跟拍或正反打
  时长：单镜头 1~2 秒
  数量：3~4 个镜头（交代情境）

【文案】
  类型：字幕 + 对白/旁白
  句式模板："那天我……""直到有一天……"
  字数范围：15~25 字
  语气：平静（故事口吻）

【声音】
  口播：必需（旁白或对白）
  BGM 情绪：温情或悬疑
  音效：环境音

【节奏】
  剪辑节奏：中速偏慢
  卡点：不需要

【降级规则】
  素材不足 MR-2 时不建议使用（故事型对素材量要求最高）。
```

### S1-F：认知颠覆型

抛出一个反常识结论，让用户想知道凭什么。

```
【适配条件】
  必需素材：T3=true, T5=true
  适配品类：通用
  排除品类：保健、母婴（反常识说法易触发违规）
  适配购买动机：MO-解决问题、MO-提升体验

【镜头】
  景别：近景（人脸）或中景（人+产品）
  主体：口播主体（最好有"专家感"或"意外感"的人）
  镜头动作：静态对视镜头
  时长：1.5~2.5 秒
  数量：1~2 个镜头

【文案】
  类型：字幕 + 口播
  句式模板："其实 XX 根本不用 XX""90% 的人都搞错了 XX"
  字数范围：10~20 字
  语气：肯定（带断言感）

【声音】
  口播：必需
  BGM 情绪：无或极轻
  音效：可选 ding 声强调

【节奏】
  剪辑节奏：慢节奏（让观点有重量）
  卡点：不需要

【降级规则】
  T3=false 或 T5=false 时不可用。
```

### S1-G：场景还原型

还原一个目标用户高度熟悉的生活场景，触发共鸣。

```
【适配条件】
  必需素材：T2≥MR-2

  适配购买动机：通用

【镜头】

  主体：日常生活场景（厨房/卧室/通勤/办公桌）
  镜头动作：静态或轻微摇移
  时长：单镜头 1~1.5 秒
  数量：2~3 个镜头

【文案】
  类型：画面文字或字幕
  句式模板："每天早上……""下班回家……"（描述场景）
  字数范围：8~15 字
  语气：平静（白描）

【声音】
  口播：可选
  BGM 情绪：轻快或温情
  音效：环境音（键盘声/煎蛋声/门铃声）

【节奏】
  剪辑节奏：中速
  卡点：可对齐 BGM

【降级规则】
  MR-1 时不建议使用，改用 S1-A 或 S1-C。
```

## 五、S2 产品引出模块库（4 个模块）

### S2-A：承接式引出

直接回应 Hook 提出的问题——"这个东西就能解决"。

```
【适配条件】
  必需素材：T5=true
  适配品类：通用
  排除品类：无
  适配购买动机：MO-解决问题（核心）、MO-刚需补货
  适配 Hook：S1-A, S1-D, S1-G

【镜头】
  景别：近景 + 中景
  主体：口播人物 + 产品（产品在口播人手中或桌面出现）
  镜头动作：静态或轻推向产品
  时长：单镜头 1~2 秒
  数量：1~2 个（口播人 1 + 产品亮相 1）

【文案】
  类型：口播 + 字幕
  句式模板："直到我发现了这个""试了 XX 之后彻底解决了""一个 XX 就搞定"
  字数范围：8~15 字
  语气：肯定（解决感）

【声音】
  口播：必需
  BGM 情绪：从 Hook 段的紧张/悬疑过渡到轻快（情绪转折点）
  音效：可选 ding 或产品出场音

【节奏】
  剪辑节奏：中速
  卡点：产品首次出现瞬间对齐情绪转折音

【降级规则】
  T5=false 时，口播替换为画面文字+产品镜头直切，仍可用但过渡感减弱。
```

---

### S2-B：解谜式引出

揭晓 Hook 里的悬念——"我用的就是它"。

```
【适配条件】
  必需素材：T6=true
  适配品类：通用
  排除品类：无
  适配购买动机：通用
  适配 Hook：S1-B, S1-C, S1-E

【镜头】
  景别：特写（产品揭晓瞬间）+ 近景
  主体：产品（从遮挡/模糊到清晰，或从画面外进入画面）
  镜头动作：快推或揭幕式运动（掀开/翻转/拆盒）
  时长：单镜头 1~1.5 秒
  数量：1~2 个（揭晓 1 + 产品正脸 1）

【文案】
  类型：画面文字
  句式模板："就是它""秘密在这里""答案揭晓"
  字数范围：3~6 字
  语气：肯定 + 揭秘感

【声音】
  口播：可选
  BGM 情绪：悬疑 → 揭晓（情绪释放点）
  音效：必需（揭幕音 / tada / 拆箱声）

【节奏】
  剪辑节奏：中速偏快
  卡点：对齐产品出现瞬间

【降级规则】
  T6=false 时，用产品包装图+画面文字替代，仍可用。
```

### S2-C：对比式引出

先否定其他方案，再引出本产品。

```
【适配条件】
  必需素材：T5=true 或 T6=true（至少一个）
  适配品类：家清日用、居家日用、厨房用品、五金工具、运动与户外
  排除品类：美妆个护（否定竞品合规风险高）、保健、珠宝与衍生品
  适配购买动机：MO-解决问题、MO-提升体验
  适配 Hook：S1-A, S1-F

【镜头】
  景别：中景
  主体：替代方案（模糊/打码）→ 本产品（清晰亮相）
  镜头动作：摇头/否定动作 → 点头/肯定动作，或画面从灰调切到亮调
  时长：单镜头 1~1.5 秒
  数量：2~3 个（否定方案 1 + 过渡 0~1 + 本品亮相 1）
  关键约束：替代方案不得出现可识别品牌 LOGO

【文案】
  类型：口播 + 画面文字
  句式模板："试过 XX、XX，都不行""别再用 XX 了""扔掉你的 XX"
  字数范围：8~15 字
  语气：否定 → 肯定（转折）

【声音】
  口播：建议有
  BGM 情绪：低沉 → 轻快（随否定到肯定转折）
  音效：否定段用 buzzer/错误音，肯定段用 ding

【节奏】
  剪辑节奏：中速
  卡点：对齐否定→肯定转折点

【降级规则】
  T5 和 T6 均=false 时，用纯画面文字+BGM 转折替代，可用但说服力弱。
```

### S2-D：第三方式引出

借朋友/医生/达人/评论之口推荐，降低广告感。

```
【适配条件】
  必需素材：T3=true
  适配品类：通用
  排除品类：无
  适配购买动机：MO-提升体验、MO-情感满足
  适配 Hook：S1-D, S1-E, S1-G

【镜头】
  景别：近景（推荐者）+ 特写（产品）
  主体：推荐者人像（朋友/家人/达人/专业人士）+ 产品
  镜头动作：推荐者面对镜头或递出产品
  时长：单镜头 1~2 秒
  数量：2~3 个（推荐者 1 + 产品亮相 1 + 互动 0~1）

【文案】
  类型：口播 + 字幕
  句式模板："朋友推荐我用的""医生/达人说 XX""评论区都在问的 XX"
  字数范围：8~15 字
  语气：平静 + 可信（转述口吻）

【声音】
  口播：必需（第三人称转述或第三方直接口播）

  音效：无或可选消息提示音

【节奏】
  剪辑节奏：中速偏慢（信任建立需要呼吸空间）
  卡点：不需要

【降级规则】
  T3=false 时，用评论截图/聊天记录截图+画面文字替代推荐者人像，仍可用但信任感下降。
```

## 六、S3 使用过程模块库（5 个模块）

### S3-A：单场景全流程

一个场景里完整展示开箱到使用。

```
【适配条件】
  必需素材：T6=true, T2≥MR-2
  适配品类：通用
  排除品类：虚拟商品
  适配购买动机：通用

【镜头】
  景别：中景为主，穿插特写（操作细节）
  主体：产品 + 使用者的手/身体 + 使用环境
  镜头动作：跟拍使用动作，关键步骤推近特写
  时长：单镜头 1.5~3 秒
  数量：4~6 个（开箱/取出 1 + 准备 1 + 使用 2~3 + 完成 1）

【文案】
  类型：字幕（步骤标注）或口播
  句式模板："打开→挤出→涂抹→等待→冲洗"（动作链）
  字数范围：每步 2~4 字，总计 10~20 字
  语气：平静（操作指引）

【声音】
  口播：可选
  BGM 情绪：轻快
  音效：ASMR 操作音（挤压/涂抹/喷洒/点击——匹配产品类型）

【节奏】
  剪辑节奏：中速
  卡点：可选对齐步骤切换

【降级规则】
  T2<MR-2 时，压缩为"取出+使用+完成"3 镜头极简版。
  T6=false 时，用产品包装中景替代特写，仍可用。
```

### S3-B：多场景拼接

同一产品在不同场景下被反复使用，强化适用面。

```
【适配条件】
