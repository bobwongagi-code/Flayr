# 目标市场知识库 1.0 - 马来西亚（东南亚泛化种子）

> 用途：分析前注入模型，仅作判断依据，不在报告中直接呈现。
>
> 结构：第一层为东南亚共性，分析任何东南亚视频都可加载；第二层为马来西亚专属，目标市场为 `my` 或存在明显马来语/马来市场信号时加载。后续印尼、泰国等市场应新增对应专属层，不要把马来专属规则泛化到所有市场。

## 第一层：东南亚共性知识

这些规律来自马来爆款拆解，作为评判东南亚视频的底层世界知识。使用时必须以视频事实为准，市场知识只用于解释某个动作在本地语境中是否可能有效。

### 1. 信任建立：要真实感，不要堆数据

东南亚 TikTok 受众偏好真实、生活化的呈现，反感啰嗦的认证、成分百分比、权威背书堆砌。

判定含义：S5 背书不是越多越好。真实使用展示优于口头权威声明；过度堆数据可能降低留存和转化。

### 2. 数量、组合、赠品代表划算和有面子

受众对数量、组合、赠品高度敏感。“东西多 = 划算 = 送人有面子 = 自己用也不亏”。`3 in 1`、`buy 1 free`、套装和大包装常是强卖点。

判定含义：套装、大包装、多功能视频，卖点维度要重点看“划算感 / 组合价值”是否传达到位。

### 3. 表达调性：亲和、轻语气、口语化

亲和、像朋友推荐的口吻，信任度和留存更高。成段大信息、推销腔、过重语气会赶走观众。

判定含义：口播是否像本地人轻松聊天，是表达调性维度的重要信号。

### 4. 节日营销与紧迫感

Raya、斋月等重大节日是消费高峰。节日前的“快卖完了”“大家都在抢”是常见且有效的促单手法。

判定含义：S6 促单维度中，节日紧迫感是正向手法，不应按中文语境误判为威胁或反感。

### 5. 本地高频 Hook

常见有效 hook 包括：视觉冲击、痛点共鸣、社会认同、情绪钩子、神奇或反常画面。

判定含义：社媒口碑放在开头时，可能是 S1 社会认同型 hook，而不是 S5 背书；归属看位置和作用。

### 6. 销售话术的本地接受度

“没买是你的损失”“快卖完了”等轻微施压话术，在本地常被理解为销售技巧，不一定引起反感。

判定含义：不要用中文审美把这类话术直接判为低级或威胁；需结合品类、语气和场景判断。

## 第二层：马来西亚专属知识

这些规则只在目标市场为马来西亚，或视频存在明确马来市场信号时使用。不可错套到其他东南亚市场。

### A. 宗教 / 族群红线

- 产品不可含猪肉或猪源成分。对马来穆斯林受众，猪肉成分是 P0 致命问题。
- 涉及 Halal、宗教圣地如 Kota Madinah 等元素时，宗教价值表述可能显著增强信任。

判定含义：发现猪肉成分等宗教红线，判 P0 致命问题，优先级高于常规 GMV 权重。

### B. 审美偏好

- 金色 + 花纹通常代表高贵、有面子，不是老土或炫富。
- 礼盒内容丰富、摆放饱满，代表值得和有诚意。
- 鲜艳、独特、闪亮颜色在节日妆容和礼品场景中常是加分项。

判定含义：金色花纹、满当摆放、亮色在马来市场可视为加分，不要按中文审美误伤。

### C. 便宜 / 省 / 方便的信任逻辑

马来受众常主动寻找便宜又方便的选择。便宜、省、方便在日用快消、猫粮等品类通常是加分点。

判定含义：不可用中文逻辑一律把便宜判为低级卖点。但需结合品类：儿童牙膏这类情绪和功能并重的产品里，“省 / 防浪费”成立，但可能弱于“香味 / 孩子爱刷”等感性卖点。

### D. 语言地道度信号

- 称呼：`korang`、`kak / akak`、`abang`、`auntie / uncle` 等用对会增强亲和。
- 口头缩写：`ni`、`tu`、`dah`、`nak`、`tak`、`je`、`kan` 等是自然本地表达。
- 马来语为主、夹少量日常英文词是自然本地表达。印度裔达人语速快、夹英文关键词，也是本地常态。

判定含义：本地称呼和缩写到位，是信任加分；不要把自然 code-switch 误判为不专业。

### E. 文化视角防误判清单

| 看到的元素 | 中文易误判 | 马来本地真实含义 |
|---|---|---|
| 卖惨的猫（耳朵没毛 / 发红） | 虐猫、卖惨博同情 | 真诚求助 + 已在行动帮助，是有效情绪钩子 |
| “没买是你损失”等施压话术 | 威胁、反感 | 常见销售技巧，无恶意 |
| 达人叹气 / 夸张表情 | 做作、假 | 真实喜爱的自然表达 |
| 画面脏乱 / 养很多猫 | 虐待、不负责 | 半散养常态，需谨慎判断 |
| 金色花纹碗碟 | 老土、炫富 | 高贵、对客人的尊重 |
| 说话特快（印度裔） | 不专业、赶 | 族群语言常态 |
| 透明收纳 / 展示食物 | 多此一举 | 美感 + 待客诚意 |

判定含义：遇到上述元素，先按本地视角理解，再判断其对转化的实际作用。
