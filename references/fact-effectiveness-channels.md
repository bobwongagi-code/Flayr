# Flayr 视频分析系统：事实提取通道完整定义 (V1.0)

> 本文档定义 Phase A（事实提取阶段）中六大事实通道的完整提取指令。
> 核心原则：**"物理存在"不等于"有效传递"**。每个通道的提取必须同时完成两层判断：
> - **存在性判定**：这个事实有没有出现？
> - **有效性判定**：这个事实有没有被消费者有效接收？

---

## **通道一：画面事实（Visual Fact）**

### 存在性判定
记录画面中实际出现的内容：产品、人物、动作、道具。

### 有效性判定——以下任一情况触发 $E$ 降级

**取景完整性**
- 关键动作是否出画（产品或核心动作超出取景框）？
- 产品主体是否完整呈现在画面中心区域？

**构图稳定性**
- 画面是否存在明显歪斜（地平线偏转超过 10 度）？
- 是否存在无意义的大幅度运镜（非创意性抖动）？

**焦点与清晰度**
- 焦点是否落在产品或关键动作上？
- 是否存在动态虚焦（拉风箱）？
- 是否存在运动模糊导致细节无法辨认？

**曝光与光影**
- 是否过曝导致产品细节（颜色、质感、文字）丢失？
- 是否逆光导致人脸或产品变成剪影？
- 是否存在强反光遮蔽关键区域（如玻璃反光、金属反光）？

**遮挡检测**
- 关键动作是否被达人自身的手、头发、道具遮挡？
- 关键内容是否落在平台 UI 安全区以外（底部 25%、右侧 15% 为危险区）？

**时长阈值**
- 单个关键动作的镜头停留时间是否低于 0.5 秒（消费者无法有效感知）？

### 确信度触发规则
- 运动模糊严重 → `confidence < 0.6`
- 关键内容落入 UI 危险区 → `confidence < 0.6`
- 动作停留时间 < 0.3 秒 → `confidence < 0.5`

## **通道二：口播事实（Voiceover Fact）**

### 存在性判定
记录口播的原始文本（Local Language 原句），不做语义修饰。

### 有效性判定——以下任一情况触发 $E$ 降级

**语速可吸收性**
- 语速是否过快导致关键信息无法被消费者正常吸收（超过正常对话语速 1.5 倍视为危险）？
- 关键卖点词是否被连读吞字？

**情绪饱和度**
- 口播情绪是否与内容匹配？（说"超级好用"但语调死平，属于声画分裂）
- 是否存在明显的"念稿感"（语调均匀无起伏、无停顿、无重音）？
- 关键卖点前是否有停顿强调（停顿是口播表现力的核心信号）？

**可听性**
- 口播是否被 BGM 或环境音压制（人声音量低于背景音）？
- 是否存在吞字、方言化导致目标用户听不懂的情况？
- 录音是否存在明显的底噪、回声或破音？

**语言匹配度**
- 口播语言是否与目标消费者的母语或习惯用语一致？
- 是否存在语言切换（如马来语切英语），切换是否自然？

### 确信度触发规则
- BGM 明显压制人声 → `confidence < 0.6`
- 转写模型原始置信度 < 0.7 → `confidence < 0.6`
- 存在大量吞字或方言 → `confidence < 0.7`

---

## **通道三：字幕与文本事实（Subtitle & Text Fact）**

### 存在性判定
记录画面中实际可读的字幕、贴纸文字、特效文字的具体内容及出现位置。

### 有效性判定——以下任一情况触发 $E$ 降级

**可读时长**
- 字幕或文字的停留时间是否低于正常阅读速度所需时间（每 5 个字符需至少 0.5 秒）？
- 关键信息是否一闪而过（< 0.3 秒）？

**视觉可辨性**
- 字体与背景的对比度是否足够（白字叠白底、黑字叠黑底均为失效）？
- 字体大小是否在移动端可读范围内（过小导致无法识别）？
- 字体风格是否过于花哨导致可读性下降？

**位置安全性**
- 文字是否落入平台 UI 危险区（底部被购物车遮挡、侧边被点赞按钮遮挡）？
- 多条字幕是否同时出现导致视觉焦点分散（信息密度过载）？

**内容与口播的一致性**
- 字幕内容是否与口播内容一致？不一致时记录差异（可能是创意强调，也可能是错误）。

### 确信度触发规则
- 文字落入 UI 危险区 → `confidence < 0.6`
- OCR 识别置信度 < 0.7 → `confidence < 0.6`
- 停留时间 < 0.3 秒 → `confidence < 0.5`

---

## **通道四：音效与 BGM 事实（Audio & BGM Fact）**

### 存在性判定
记录视频中是否存在 BGM、音效，以及其类型（欢快/悲伤/紧张/无）。

### 有效性判定——以下任一情况触发 $E$ 降级

**BGM 情绪匹配度**
- BGM 的情绪基调是否与视频内容和品类调性一致？
  - 清洁/功能品类配悲伤慢歌 → 情绪错配，拉低信任感
  - 儿童产品配紧张音乐 → 情绪错配，引发焦虑
- BGM 节奏是否与画面剪辑节奏匹配（音画不同步会产生割裂感）？

**音效时机准确性**
- 产品相关音效（开盖声、喷射声、点击声）是否与对应画面动作同步？
- 音效延迟超过 0.3 秒视为时机错位，降低真实感。

**音量层级关系**
- BGM 是否喧宾夺主（音量高于人声）？
- 关键音效是否被 BGM 掩盖（如产品使用的声音细节）？

**静音段落**
- 在关键表达阶段（S1 Hook、S4 效果呈现、S6 CTA）是否出现非创意性静音段落？
- 非创意性静音会导致消费者注意力断裂。

### 确信度触发规则
- BGM 明显压制人声 → 同时触发口播通道低置信度
- 音效与画面明显不同步 → `confidence < 0.7`

## **通道五：场景事实（Scene Fact）**

### 存在性判定
记录拍摄的物理环境（如：真实家居、浴室、厨房、影棚白底、户外街头）。

### 有效性判定——以下任一情况触发 $E$ 降级

**真实感**
- 场景是否具有真实生活感？影棚白底、过度布景会降低消费者代入感。
- 场景道具是否与产品使用逻辑一致（如清洁剂出现在干净整洁的环境里，无法体现痛点）？

**目标人群匹配度**
- 场景是否与目标消费者的真实生活环境匹配？
  - 豪华别墅场景推销 9.9 元日用品 → 人群脱节
  - 马来西亚本地产品出现明显中国场景元素 → 文化错位

**场景停留时长**
- 单个场景的停留时间是否足够让消费者建立代入感（< 1 秒的场景切换无法建立情境）？
- 是否存在场景切换过于碎片化（单个场景平均时长 < 0.8 秒）？

**场景与阶段功能的匹配**
- S3（使用过程）的场景是否真实还原了产品的使用环境？
- S4（效果呈现）的场景是否支撑了效果对比的可信度？

### 确信度触发规则
- 场景停留时间 < 0.5 秒 → `confidence < 0.6`
- 场景与产品使用逻辑明显矛盾 → 标注冲突信号，触发 Phase C

## **通道六：背书事实（Endorsement Fact）**

### 存在性判定
记录画面中实际出现的第三方证明：认证标志、检测报告、用户评价截图、专业人员出镜、销量数据。

### 有效性判定——以下任一情况触发 $E$ 降级

**可识别时长**
- 认证标志、检测报告等背书内容的停留时间是否足够消费者识别（至少 1.5 秒）？
- 一闪而过的认证标志（< 0.5 秒）等同于无效背书。

**画面清晰度**
- 认证标志、报告文字是否清晰可读？像素不足或拍摄角度倾斜导致无法辨认视为无效。
- 是否存在反光或遮挡导致关键信息不可读？

**孤证检测**
- 是否只有口播提及背书，而画面没有任何视觉佐证？
- 孤证（只有口播，无画面）在高决策门槛品类中视为无效背书，降级为 $E=1$。

**背书与产品的关联性**
- 背书内容是否与该产品直接相关（通用认证不等于产品专属认证）？
- 用户评价截图是否清晰显示了与该产品相关的内容？

### 确信度触发规则
- 背书内容停留时间 < 0.5 秒 → `confidence < 0.5`
- 只有口播无画面佐证 → `confidence < 0.6`，同时标注孤证风险

## **附录：六通道有效性判定速查表**

| 通道 | 物理存在 | 典型失效模式 | $E$ 降级触发 |
|---|---|---|---|
| **画面** | 动作/产品出现了 | 出画、遮挡、虚焦、歪斜、UI 遮挡 | 任一情况 → $E$ 最多为 1 |
| **口播** | 说了卖点 | 语速过快、念稿、被 BGM 盖过、吞字 | 任一情况 → $E$ 最多为 1 |
| **字幕** | 文字出现了 | 停留太短、UI 遮挡、对比度不足、密度过载 | 任一情况 → $E$ 最多为 1 |
| **音效/BGM** | 有音乐/音效 | 情绪错配、时机错位、音量压制人声 | 任一情况 → $L_{consumer}$ 下调 |
| **场景** | 有拍摄背景 | 影棚感、人群脱节、切换太快、逻辑矛盾 | 任一情况 → $E$ 最多为 1 |
| **背书** | 提到了认证 | 一闪而过、只有口播无画面、像素不足 | 任一情况 → $E$ 最多为 1 |

## **附录：Phase C 触发优先级**

当多个通道同时出现低置信度信号时，按以下优先级决定回看顺序：

| 优先级 | 触发条件 | 原因 |
|---|---|---|
| **P1（必看）** | S1/S6 阶段任一通道 `confidence < 0.6` | 链路致命节点，判错代价最高 |
| **P2（高优）** | S4 效果呈现阶段声画冲突 | 高杠杆说服节点，孤证风险大 |
| **P3（常规）** | 任一通道 $S$ 分值处于临界区间（2.4\~2.6） | 防止边界判定失准 |
| **P4（备选）** | 模型自述 `certainty_score < 0.7` | AI 自知不确定，主动申请复核 |
